# Flow Hub v1.0.0

Flow Hub 的首个 Qwen 驱动正式版本。

这个版本完成了桌面端 Agent 驱动从 iFlow CLI 到 Qwen CLI 的切换，并补齐了 Qwen 历史会话、工作区文件列表、长任务活动可见性等核心体验。

## 亮点

- 全量切换到 Qwen CLI + ACP stdio 驱动
- 支持导入 Qwen 历史会话
- 新增工作区文件列表面板
- 长任务期间可显示最近活动时间、当前阶段、最近工具
- 修复 HTML 预览路径识别异常、上下文占比显示精度、历史会话重复等问题

## 重要变更

- 这是 `1.0.0` 正式版本，标志着 Flow Hub 进入 Qwen 驱动阶段
- 运行时连接方式已从旧的 WebSocket ACP 调整为 stdio ACP
- 模型与能力信息优先依赖 ACP 运行时返回，而不是旧的静态解析链路

## 安装包

- macOS Apple Silicon: `flow hub_1.0.0_aarch64.dmg`

## 升级说明

- 当前仍保留 `com.iflow.workspace` bundle identifier 与部分 `iflow-*` 本地存储 key，以保证旧版本升级路径和已有本地数据兼容
- 已有用户可直接升级到该版本，无需手动迁移本地历史数据
